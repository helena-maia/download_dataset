- Create virtual environment with the desired python
    1. Check python directory
        which python[<version>]

    2. Create virtualenv
        virtualenv --python='<python_dir>' <name>

    3. Activate
        source <name>/bin/activate

- Using the virtualenv (activated)
    1. Copy src directory
        cp <path_to_src>/src <path_to_virtualenv>/<name>
    2. Install requirements
        pip install -r requirements.txt
    3. Check packages
        pip list
    4. Check python version
        python --version

- Deactivate and remove virtual environment
    1. Deactivate
        deactivate 
    2. Remove
        rm -r <name>
